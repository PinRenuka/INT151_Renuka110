วันนี้ทบทวนไปหลายเรื่องเพราะต้องทำการบ้านมีการทวนพวก method ของ Array String เพิ่มด้วย
แล้วก็การเขียน if else loop เพิ่มเติม รู้สึกว่าพวก method จำแทบจะไม่ได้เลยจำได้แต่พวกที่ใช้บ่อย 
ในการบ้านวันนี้ได้ใช้ .trim() ไปตอนแรกก็จำไม่ได้เลยไปนั่งเรียนกับวิดีโอที่สอนในห้องไปก็มีพูดถึงอันนี้
แต่ไม่ได้ยกตัวอย่างให้ดูก็เลยไปถามความหมายวิธีการใช้ดูกับ AI มันก็ให้คำตอบแล้วก็ตัวอย่างให้ดูก็เข้าใจมากขึ้น
ส่วน method อื่นก็จำได้อยู่ว่าใช้ยังไงมันคืออะไร ในส่วนของ if ก็คือมีหลายเงื่อนใขมากจากในโจทย์ที่ให้มาก็ว่าจะลองใช้ else-if 
แต่จะเขียนยาวไปเลยใช้ if แบบเขียนยาวๆแล้วเอาพวก Operator || ช่วยด้วย ส่วนของ loop อันนี้ใช้เป็นอยู่แล้วแต่ก็ทวนใหม่เพื่อความเข้าใจมากขึ้น
